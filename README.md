# Summarist - Book Summaries at Your Fingertips

## Live Website
🌐 **https://summarist-ten-plum.vercel.app**

A modern web application that provides book summaries in audio and text format. Built with Next.js, TypeScript, and Tailwind CSS.

## Features

- 📚 **Book Summaries** - Read or listen to book summaries
- 🔐 **Authentication** - Login/Register with email or as guest
- 🎵 **Audio Player** - Listen to book summaries with a custom audio player
- 💳 **Subscriptions** - Premium plans with Stripe integration
- 🔍 **Search** - Search books by title or author
- 📱 **Responsive** - Works on desktop and mobile

## Tech Stack

- **Framework**: Next.js 16 with App Router
- **Language**: TypeScript
- **Styling**: CSS
- **State Management**: Redux Toolkit
- **Authentication**: Firebase
- **Payments**: Stripe
- **Icons**: React Icons

## Deployment

Deployed on Vercel: https://summarist-ten-plum.vercel.app
